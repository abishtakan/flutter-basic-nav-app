# Flutter Basic Navigation App with Input and Display

This Flutter application demonstrates basic navigation along with input handling and data passing between screens. The app includes a splash screen, a home page, a second page with a text input field, and a third page that displays the entered message.

## Features

- **Splash Screen**: A simple splash screen appears on app launch, transitioning to the home page after a short delay.
- **Home Page**: The main screen of the app, from which users can navigate to the second page.
- **Second Page**: Allows users to enter a message in a text field and navigate to the third page.
- **Third Page**: Displays the message entered on the second page.

## Getting Started

To run this app on your local machine, follow these steps.

### Prerequisites

Ensure Flutter is installed on your system. If not, follow the instructions on the [Flutter official website](https://flutter.dev/docs/get-started/install) to install Flutter.

### Installation

1. Clone the repository to your local machine:
   ```sh
   git clone https://github.com/abishtakan/flutter-basic-nav-app.git
   ```
2. Navigate to the project directory:
   ```sh
   cd flutter_navigation_app
   ```
3. Fetch the required Flutter packages:
   ```sh
   flutter pub get
   ```
5. Run the app:
   ```sh
   flutter run
   ```
